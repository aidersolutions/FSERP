Let’s cancel the shipment and handle the issue from the purchase. As a consequence the sale order is over:

>>> config.user = stock_user.id
>>> shipment.click('cancel')

>>> config.user = purchase_user.id
>>> purchase.reload()

u'exception'
>>> handle_exception = Wizard('purchase.handle.shipment.exception',
...     [purchase])

>>> purchase.reload()
>>> purchase.shipment_state
u'received'

>>> config.user = sale_user.id
>>> sale.reload()
>>> sale.shipment_state
u'sent'
